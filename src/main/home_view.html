WELCOME TO THE BRONCO STUDENT HUB

Goal of this Site

This was built as a project for CS4080, using TypeScript, CSS, and HTML.

COVID has made it very difficult for students to find a way to stay updated in the classes and find ways to connect and find help. This site is meant to be a useful hub where students can check their grade statuses (especially when blackboard doesn't count the weighted grades), check their GPA and Graduation statuses, ask questions that other students may be able to answer, as well as find many other CPP resources all in one place.

Goals for the Future of this Site

Since TypeScript supports large scale projects, the future of this app would include integrating a database to save usere's data, so that the website may function much more efficiently for the users.
